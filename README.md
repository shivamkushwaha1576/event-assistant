# 🚀 AI Powered Smart Event Assistant

## 📌 Challenge Vertical
Physical Event Experience

During physical events (college fests, seminars, concerts), users often get confused about which sessions to attend based on their interests and time availability.

---

## 💡 Solution
This project is a smart web-based assistant that helps users choose the best events by analyzing their interest and available time.

---

## ⚙️ How It Works
1. User enters their name  
2. Selects interest (Tech, Music, Sports)  
3. Enters available time  
4. System filters relevant events  
5. Suggests the best event + other options  
6. User can set a reminder  

---

## 🚀 Features
- 👤 Personalized greeting  
- 🧠 Smart event filtering logic  
- 🔥 Best event recommendation (priority-based)  
- 📋 Multiple event suggestions  
- ⏰ Reminder feature  



## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## 📌 Assumptions
- User inputs valid data  
- Event list is predefined  
- Reminder is simulated using browser alerts
